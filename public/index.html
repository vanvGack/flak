Users

POST /users

Create a new user account.

Request:

{
  "user" => {
    "first_name": "Name",
    "last_name": "Name",
    "email": "email@example.com",
    "password": "asdfasdf",
    "password_confirmation": "asdfasdf"
  }
}

GET /users

Retrieve a list of registered users.

GET /users/current

Retrieve a list of currently logged in users.

Sessions

POST /session

Create a new user session.

Request:

{
  "session" => {
    "email" => "email@example.com",
    "password" => "asdfasdf"
  }
}

DELETE /session

Destroy the current user session.

Messages

GET /messages

Get a list of messages.

POST /messages

Post a new message.

Request:

{
  "message" => {
    "body" => "email@example.com"
  }
}
